Générer une nouvelle bibliothèque de prompts

Utilisez le prompt suivant pour générer une bibliothèque de prompts pouvant être importée directement dans ce plugin.

Générez 20 des prompts les plus efficaces et de la plus haute qualité que vous avez récemment rencontrés ou créés. Chaque prompt doit être particulièrement perspicace, créatif ou éducatif, utile pour l’apprentissage ou l’inspiration.

Fournissez chaque résultat selon la structure JSON suivante :

{
  "title": "Short descriptive title of the prompt",
  "content": "Full text of the prompt. Use {variables} for any placeholders.",
  "category": "General theme or purpose of the prompt (e.g., Writing, Analysis, Creativity)",
  "tags": ["tag1", "tag2"],
  "variables": ["{variables used}"],
  "isFavorite": false,
  "useCount": 0,
  "lastUsed": null
}

Assurez-vous que chaque objet JSON soit autonome, clairement rédigé et formaté pour un parsing facile.
