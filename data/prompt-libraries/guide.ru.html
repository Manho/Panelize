Создать базовую библиотеку промптов

Три самых мощных базовых промпта

  • Перепишите следующий промпт на ясный, естественный русский язык и улучшите его, применяя лучшие практики написания промптов для ИИ, чтобы получить более точные, конкретные и полезные результаты. Промпт: {draft}
  • Перечислите как можно больше релевантных пунктов, примеров или идей по следующей теме или вопросу. Стремитесь к полноте, точности и ясности. Тема: {topic}
  • Какие фреймворки, методы или модели наиболее эффективны для {objective}? Для каждого фреймворка объясните его основную идею, преимущества и когда его лучше применять.

Вы можете импортировать их по пути к файлу: data/prompt-libraries/three-samples.json

Создать новую библиотеку промптов

Используйте следующий промпт, чтобы сгенерировать библиотеку промптов, которую можно напрямую импортировать в этот плагин.

Сгенерируйте 20 самых эффективных и качественных промптов, которые вы недавно встречали или создавали. Каждый промпт должен быть особенно содержательным, креативным или образовательным — ценным для обучения или вдохновения.

Предоставьте каждый результат в следующей структуре JSON:

{
  "title": "Short descriptive title of the prompt",
  "content": "Full text of the prompt. Use {variables} for any placeholders.",
  "category": "General theme or purpose of the prompt (e.g., Writing, Analysis, Creativity)",
  "tags": ["tag1", "tag2"],
  "variables": ["{variables used}"],
  "isFavorite": false,
  "useCount": 0,
  "lastUsed": null
}

Убедитесь, что каждый объект JSON самодостаточен, четко написан и отформатирован для удобного парсинга.
