기본 프롬프트 라이브러리 만들기

가장 강력한 기본 프롬프트 3가지

  • 다음 프롬프트를 명확하고 자연스러운 한국어로 다시 작성하고, AI 프롬프트 작성 모범 사례를 적용해 더 정확하고 구체적이며 유용한 결과를 얻도록 개선하세요. 프롬프트: {draft}
  • 다음 주제나 질문과 관련된 항목, 예시, 아이디어를 가능한 한 많이 나열하세요. 완전성, 정확성, 명확성을 목표로 합니다. 주제: {topic}
  • {objective}에 가장 효과적인 프레임워크, 방법 또는 모델은 무엇인가요? 각 프레임워크의 핵심 아이디어, 장점, 그리고 가장 적합한 사용 상황을 설명하세요.

다음 파일 경로로 이 프롬프트들을 가져올 수 있습니다: data/prompt-libraries/three-samples.json

새 프롬프트 라이브러리 생성

다음 프롬프트를 사용해 이 플러그인에 바로 가져올 수 있는 프롬프트 라이브러리를 생성하세요.

최근에 접하거나 만든 가장 효과적이고 고품질의 프롬프트 20개를 생성하세요. 각 프롬프트는 통찰력, 창의성 또는 교육적 가치가 있어 학습이나 영감에 도움이 되어야 합니다.

각 결과를 다음 JSON 구조로 제공하세요:

{
  "title": "Short descriptive title of the prompt",
  "content": "Full text of the prompt. Use {variables} for any placeholders.",
  "category": "General theme or purpose of the prompt (e.g., Writing, Analysis, Creativity)",
  "tags": ["tag1", "tag2"],
  "variables": ["{variables used}"],
  "isFavorite": false,
  "useCount": 0,
  "lastUsed": null
}

각 JSON 객체가 자체적으로 완결되고, 명확하게 작성되며, 쉽게 파싱될 수 있도록 해주세요.
