Generare una nuova libreria di prompt

Usa il seguente prompt per generare una libreria di prompt che può essere importata direttamente in questo plugin.

Genera 20 dei prompt più efficaci e di alta qualità che hai incontrato o creato recentemente. Ogni prompt deve essere particolarmente perspicace, creativo o educativo, utile per l'apprendimento o l'ispirazione.

Fornisci ogni risultato con la seguente struttura JSON:

{
  "title": "Short descriptive title of the prompt",
  "content": "Full text of the prompt. Use {variables} for any placeholders.",
  "category": "General theme or purpose of the prompt (e.g., Writing, Analysis, Creativity)",
  "tags": ["tag1", "tag2"],
  "variables": ["{variables used}"],
  "isFavorite": false,
  "useCount": 0,
  "lastUsed": null
}

Assicurati che ogni oggetto JSON sia autonomo, scritto chiaramente e formattato per una facile analisi.
