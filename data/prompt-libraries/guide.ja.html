新しいプロンプトライブラリを生成

以下のプロンプトを使用して、このプラグインに直接インポートできるプロンプトライブラリを生成してください。

最近見つけたり作成したりした最も効果的で高品質なプロンプトを20件生成してください。各プロンプトは特に洞察的で、創造的または教育的であり、学習やインスピレーションに役立つものにしてください。

各結果を次のJSON構造で提示してください:

{
  "title": "Short descriptive title of the prompt",
  "content": "Full text of the prompt. Use {variables} for any placeholders.",
  "category": "General theme or purpose of the prompt (e.g., Writing, Analysis, Creativity)",
  "tags": ["tag1", "tag2"],
  "variables": ["{variables used}"],
  "isFavorite": false,
  "useCount": 0,
  "lastUsed": null
}

各JSONオブジェクトが自己完結し、明確に書かれ、解析しやすい形式であることを確認してください。
